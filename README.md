# Repositório de Atividades - Disciplina Optativa de IoT

Este repositório reúne as atividades desenvolvidas na disciplina optativa de Internet das Coisas (IoT) do curso de Engenharia de Computação do IFPB - Campus Campina Grande (IFPB-CG).

## Objetivo

Centralizar as práticas da disciplina em um único repositório, permitindo:

- organização das entregas por atividade;
- versionamento do código e documentação;
- reprodução dos experimentos em outras máquinas;
- acompanhamento da evolução técnica ao longo da disciplina.

## Organização do Repositório

As atividades devem ser organizadas em pastas, por exemplo:

```text
Atividade_1/
Atividade_2/
Atividade_3/
...
```

Cada atividade deve conter, no mínimo:

- código-fonte;
- arquivo de descrição da atividade (README da pasta);
- instruções de compilação e execução.

## Tecnologias Utilizadas

- ESP32
- ESP-IDF
- C / CMake
- Git e GitHub

## Boas Práticas de Entrega

- não versionar arquivos gerados automaticamente (ex.: build, logs);
- manter arquivos de configuração e código-fonte necessários para compilar em outra máquina;
- documentar dependências, passos de build e execução;
- usar commits descritivos para cada etapa da atividade.

## Como Utilizar

1. Clone este repositório.
2. Entre na pasta da atividade desejada.
3. Siga o README específico da atividade para compilar e executar.
